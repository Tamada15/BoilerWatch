
## WPF-приложение для удаленного мониторинга параметров котельных через Owen Cloud API.
*   **Язык:** C#
*   **Фреймворк:** .NET 9, WPF
*   **Архитектура:** MVVM (Community Toolkit MVVM)
*   **Графики:** LiveCharts2
*   **HTTP-клиент:** REST API
*   **Дополнительно:** JSON, LINQ, MahApps
